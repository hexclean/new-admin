Move HTML element using jQuery

jQuery has a function that allows you to select 1 or more HTML elements and append them to a different HTML element. You identify the element you want to move then use the .appendTo method and in the parenthesis you identify the element you want it moved to. This function was created inside of a click event of the button under the two boxes. When you click the button, you will see the "My Target 1 item" move to the rightDiv and append itself to the bottom of the other items already in that area.

$(document).ready(function() { $("#eventBtn").click(function(){ $("#target1").appendTo("#rightDiv"); }); });
